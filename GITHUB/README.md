# GitHub Repositories Documentation

This directory contains documentation and information about my GitHub repositories.

## Repository List

1. [KDE-CachyOS](https://github.com/DUCKMAN2016/KDE-CachyOS)
   - Purpose: Backup and documentation of KDE Plasma configuration on CachyOS
   - Contents:
     - KDE configuration files
     - Wallpaper settings
     - System settings
     - Application configurations
     - Lock screen configuration (Anime-Room background)

2. [Hyprland Configuration]{Not Currently Using}(https://github.com/DUCKMAN2016/hyprland-config)

## Repository Organization

- Each repository is focused on a specific aspect of the system configuration
- Configuration files are organized by category and purpose
- Backup of system-wide settings including lock screen configuration
- Configuration files are organized by category
- Documentation is included for each major component
- Personal data is removed from configurations
- Binary files (wallpapers, etc.) are stored separately when possible

## Security Considerations

- No personal data is stored in repositories
- IP addresses and network configurations are not included
- Sensitive information is removed from configuration files
- Environment variables are used instead of hardcoded paths

## Usage

1. Clone the repository you need
2. Review the README in each repository for specific setup instructions
3. Use the configuration files as templates or direct replacements
4. Refer to the documentation for any custom modifications needed

## Contributing

- Fork the repository if you want to contribute
- Create a feature branch
- Submit a pull request with your changes
- Include documentation for any new features
