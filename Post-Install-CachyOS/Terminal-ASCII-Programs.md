# Terminal ASCII Art Programs

This document describes the ASCII art entertainment programs that are installed as part of the post-install script.

## cmatrix

A curses-based scrolling 'Matrix'-like screen.

- **Usage**: `cmatrix`
- **Description**: Displays animated falling characters reminiscent of the Matrix movie
- **Package**: Available in official repositories

## asciiquarium

An aquarium/sea animation in ASCII art.

- **Usage**: `asciiquarium`
- **Description**: Shows animated fish swimming in an ASCII aquarium
- **Package**: Available in official repositories

## bonsai.sh

A bash script that generates beautiful bonsai trees in the terminal.

- **Usage**: `bonsai.sh`
- **Description**: Creates ASCII art bonsai trees that grow over time
- **Installation**: Downloaded from GitLab and installed to /usr/local/bin
- **Source**: https://gitlab.com/jallbrit/bonsai.sh
